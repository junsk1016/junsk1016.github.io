## 11720번 **숫자의 합**
---

### 문제
---
  N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

#### 입력
첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.
```
5
54321
```

#### 출력
입력으로 주어진 숫자 N개의 합을 출력한다.
```
15
```

#### 정답
```c++
#include <iostream>
using namespace std;

int main()
{

	int n;
	int sum = 0;

	cin >> n;

	char * str = new char[n + 1];
	cin >> str;

	for (int i = 0; i < n; i++)
	{

		sum += str[i] - '0'; //문자열을 숫자로 바꾸기 위함. 아스키코드를 이용하였다. 0은 아스키 코드에서 48
	}

	cout << sum << endl;

	return 0;
}
```
